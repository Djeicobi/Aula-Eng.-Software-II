Projeto: E-commerce de fotografias

Titulo: Foto Síntese

1.Análise de Requisitos:
 
* Sistema de e-commerce de fotografias digitais, baseada em demandas ou amostras gerais;

* Requisitos:
 Requisitos Funcionais:
	O sistema deve cadastrar usuários (cliente);
	O sistema deve cadastrar eventos;
	O sistema deve armazenar fotos (arquivos de imagem) dentro desses eventos;
	O sistema deve armazenar em um carrinho de compras, as fotos selecionadas pelos usuários;
	O sistema deve disponibilizar as imagens adquiridas pelo usuário em uma página, além de oferecer o envio por e-mail;
	O sistema deve disponibilizar a contratação de eventos por clientes (usuários);
Requisitos Inversos:
	O sistema não deve permitir a aquisição da mesma foto mais de uma vez, pelo mesmo usuário;
	O sistema não deve permitir o download das imagens sem a aquisição prévia;
